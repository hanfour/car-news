# 文章結構模板

## 標題
- 長度：15-25字
- 格式：[品牌/事件] + [核心事實] + [關鍵數據]
- ✅ 範例：「特斯拉 Model 3 降價 30%，美國市場 5 月生效」
- ❌ 反例：「震驚！特斯拉瘋狂大降價」

## 正文結構

**重要：以下是內部結構指引，不要在正文中加入任何段落標題或編號！**
**文章應該是連貫的純文字敘述，自然分段，不要有「導語」「背景分析」「多元視角」等標題！**

### 開頭段（100-150字）
- 用2-3句話概括核心事實
- 包含：誰（Who）、做了什麼（What）、何時生效（When）、關鍵數據（How much）
- 直接進入主題，不要寫「第一段」或「導語」

### 中間段（200-300字）
- 解釋「為什麼重要」
- 包含：產業背景、歷史對比、影響範圍
- 自然銜接，不要寫「第二段」或「背景分析」

### 補充段（選用，100-200字）
- 如果來源有不同說法或爭議，並列呈現
- 自然融入敘述，不要寫「第三段」或「多元視角」

### 結尾：來源標註（固定格式）
```
---
**資訊來源**
- [來源1標題](url1)
- [來源2標題](url2)

*本文綜合多方報導撰寫，力求客觀呈現事實全貌。*
```

## ❌ 絕對禁止
- 不要輸出「第一段：」「第二段：」「第三段：」
- 不要輸出「導語」「背景分析」「多元視角」等標題
- 不要在段落前加任何標題或編號
- 正文應該是連貫的敘述文章

## 長度控制
- 總字數：600-900字
- 開頭段：100-150字
- 中間段：200-300字

## 數據標註規範
- 價格：必須註明貨幣單位和市場
- 日期：完整日期格式
- 百分比：保留一位小數
