# 文章結構模板

## 標題
- 長度：15-25字
- 格式：[品牌/事件] + [核心事實] + [關鍵數據]
- ✅ 範例：「特斯拉 Model 3 降價 30%，美國市場 5 月生效」
- ❌ 反例：「震驚！特斯拉瘋狂大降價」

## 正文結構

### 第一段：導語（100-150字）
目的：用2-3句話概括核心事實
必須包含：誰（Who）、做了什麼（What）、何時生效（When）、關鍵數據（How much）

### 第二段：背景分析（200-300字）
目的：解釋「為什麼重要」
必須包含：產業背景、歷史對比、影響範圍

### 第三段：多元視角（選用，100-200字）
目的：如果來源有不同說法或爭議，並列呈現

### 結尾：來源標註（固定格式）
```
---
**資訊來源**
- [來源1標題](url1)
- [來源2標題](url2)

*本文綜合多方報導撰寫，力求客觀呈現事實全貌。*
```

## 長度控制
- 總字數：600-900字
- 導語：100-150字
- 背景分析：200-300字

## 數據標註規範
- 價格：必須註明貨幣單位和市場
- 日期：完整日期格式
- 百分比：保留一位小數
