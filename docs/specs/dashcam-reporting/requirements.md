# 行车记录车祸回报系统 - Requirements Document

让用户上传行车记录，用于即时车祸回报。串接 Google Maps / Apple Maps 显示车祸路段，让用车人知道哪里发生车祸。路段上的车主可上传行车记录作为证据，提供给车祸当事人或警察机构。这是项目战略转型的核心功能。
